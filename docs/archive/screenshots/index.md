# Screenshot Archive

Use this folder for milestone screenshots, before-and-after captures, or other visual checkpoints that help explain meaningful UX changes.

Good uses:

- baseline captures before a workflow refactor
- after-state captures that show a meaningful UX improvement
- evidence for layout, density, contrast, or responsive drift
- screenshots referenced by manual testing or workflow reviews

Suggested filename pattern:

- `YYYY-MM-DD-area-scenario-state.png`

Suggested companion notes:

- scenario or path represented
- viewport or device
- why the capture is worth keeping
- what later comparisons should look for

Keep only captures that support product, UX, or review history.
