## Design

### Intent

Convert the distilled project understanding into the first real implementation-ready arc.

### User or system outcome

- the phase should define which paths and validations matter

### Definition of done

- the first real arc exists
- the first active phase is implementation-ready
- change-map rows and QA commands reflect the real stack
