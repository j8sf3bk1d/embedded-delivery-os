# PRD v0 Intake

Use this folder for imported product requirements, copied chats, exported docs, brainstorm notes, and other early source material.

If no formal PRD exists yet, this is also the default location for the first durable discovery brief created from live discussion.

Rules:

- preserve raw imports for provenance
- do not treat every imported file as equal authority by default
- update `docs/system/source-authority.md` after adding source material
- create a derived working distillation if the corpus becomes dense
- if discovery starts from conversation, create `10-discovery-brief.md` before planning implementation

Recommended starter contents:

- `10-discovery-brief.md` when no PRD exists yet
- `10-product-requirements-v0.md` when a PRD already exists
- `20-scope-and-non-goals.md`
- `30-landmine-map.md`
- `40-test-matrix.md`
- `README.md` describing the pack if it was imported from elsewhere
