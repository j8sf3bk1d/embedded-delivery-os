# Tutorial Build Layer

This folder exposes the source structure used to "compile" the public tutorial lessons.

The final lessons should live closer to the root of `docs/tutorial/`.

This build layer exists so the tutorial can be developed intentionally instead of drifting into a pile of disconnected essays.

## What Lives Here

- audience model
- framework inventory
- failure-family map
- story bank
- lesson requirements
- tutorial architecture and build plan

## Usage Rule

Use this folder to plan, map, and refine tutorial content.

Do not treat these files as the public front door. They are the source materials and compilation notes behind the final lessons.
